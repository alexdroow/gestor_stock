Este panel es referencial y se calcula con tus ventas semanales + facturas cargadas. No reemplaza la declaracion oficial en SII ni revision contable.
Como rellenar la tabla SII (guia rapida)
1) Ventas brutas Ingresa el total vendido del mes (local + apps) con IVA incluido.
2) Compras sin IVA Ingresa base imponible de compras. Si solo tienes monto con IVA, usa columna referencia para dividir por 1 + IVA.
3) IVA debito / credito Puedes ajustarlos manualmente. IVA neto se calcula automaticamente para evitar errores.
Formulas usadas por el asistente: Comision apps = (Uber + PedidosYa) x % comision, IVA neto = max(0, IVA debito - IVA credito), Resultado operacional = Ventas netas s/IVA - Compras s/IVA.
